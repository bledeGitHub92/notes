## 标识符

下划线、美元符号、字母、数字组成。不能以数字开头。

## 数字

NaN 是一个不等于任何值，包括自己的数字。

可用 isNaN() 检测。

## 字符串

字符串一旦创建，就不可改变。

两个包含着完全相同的字符且字符顺序也相同的字符串是相同的字符串。

    'c' + 'a' + 't' = 'cat';

## 语句

一个编译单元包含一组可执行的语句。

在 web 浏览器中，每个 < script > 标签提供一个被编译且立即执行的编译单元。

switch、while、for、do 语句允许有一个可选的前置标签（label:），它配合 break label 使用。

    foo: for (let i = 0; i < 10; i++) {
        console.log(i);
        if (i === 8) {
            break foo;
        }
    }

下面列出的值被当作假

- 0、NaN
- ''
- false
- undefined
- null

## 表达式

最简单的表达式：

- 字面量（如字符串或数字）
- 变量
- 内置值（true、false、null、undefined、NaN、Infinity）
- 以 new 开头的调用表达式
- 以 delete 开头的属性提取表达式
- 包在圆括号中的表达式
- 以一个前置运算符作为前导的表达式
- 表达式后面跟着：
    - 一个中置运算符和另外一个表达式
    - 三元运算符 ? 后面跟着另外一个表达式，然后接一个 ：，再接第三个表达式
    - 一个函数调用
    - 一个属性提取表达式

表达式优先级：

| . [] ()                 | 提取属性与调用函数  |
|-------------------------|------------|
| delete new typeof + - ! | 一元运算符      |
| * / %                   | 乘法、除法、求余   |
| + -                     | 加法 / 连接、减法 |
| >= <= > <               | 不等式运算符     |
| === !==                 | 等式运算符      |
| &&                      | 逻辑与        |
| II                      | 逻辑或        |
| ?:                      | 三元         |

typeof 返回的值有 number、string、boolean、undefined、function、object
